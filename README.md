安装
------------

The preferred way to install this extension is through [composer](http://getcomposer.org/download/).

Either run

```
git clone https://github.com/ice-leng/hyperf-advanced-frontend.git
```


运行
----
```
yarn install
```

- 开发模式运行
```
yarn run serve
```

- 编译项目
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
